# Omer's Clock

A simple Minecraft mod built with the **Fabric** modding toolchain for version **1.21.11**. It adds a real-world clock HUD to your game, so you can keep track of time without leaving your world.

## Features
- **Real-World Clock HUD:** Displays the current time (HH:mm:ss) in the top-left corner of the screen.
- **Sleek Design:** Features a translucent background for better readability.
- **Toggle Command:** Use `/clock-toggle` to show or hide the clock at any time.

## Installation
1.  Ensure you have **Fabric Loader** installed for Minecraft **1.21.11**.
2.  Download the latest version of **Fabric API** and place it in your `mods` folder.
3.  Place the `omers-clock.jar` into your `mods` folder.
